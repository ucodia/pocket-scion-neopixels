# pocket-scion-neopixels

Guide for externalizing the NeoPixel output of the [Instruo Pocket Scion](https://www.instruomodular.com/product/pocket-scion/) to drive an off-board LED strip.

## What this is

The Pocket Scion has onboard NeoPixels that respond to bioelectric input. This guide shows how to tap the NeoPixel data line and extend the visualization to a separate LED strip, powered via USB.
