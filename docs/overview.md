# Overview

## What you're doing

The Pocket Scion has onboard NeoPixels that respond to bioelectric input. This guide shows how to tap that same data signal and drive an external 9-LED strip, so the light show is bigger and visible from a distance.

## Parts

| Part | Qty | Notes |
|---|---|---|
| Pocket Scion | 1 | Provided |
| Individually-addressable NeoPixel LED | 9 | Any of these form factors:<br>• [NeoPixel Diffused 8mm Through-Hole (#1734)](https://www.adafruit.com/product/1734)<br>• [NeoPixel Diffused 5mm Through-Hole (#1938)](https://www.adafruit.com/product/1938)<br>• [Flora RGB Smart NeoPixel v3, sheet of 20 (#1559)](https://www.adafruit.com/product/1559) |
| 1N4001 silicon diode | 1 | [Adafruit #755](https://www.adafruit.com/product/755). Any 1N400x works. |
| 22 AWG solid-core wire | ~2m | [Adafruit #290](https://www.adafruit.com/product/290) |
| USB-A cable (sacrificial, will be cut) | 1 | Must be USB-A. To use USB-C instead, see the [Adafruit USB-C Breakout #4090](https://www.adafruit.com/product/4090) alternative. |
| USB-C cable | 1 | For powering the Scion |
| Dual-port USB wall charger or power bank | 1 | Must be a single unit with two ports |
